FAQ

Common questions

Everything important is now built into the page.

How does payment work?
All main payment buttons on this page send visitors directly to your Stripe payment link for checkout.
What happens after someone pays?
After checkout, Stripe should redirect them back to this page using the success URL. A success notice appears, and they can immediately complete the project brief form.
Where do project details get sent?
The “Send by Email” button opens the visitor’s email app and sends the project brief to Sevada.Zadooryan@gmail.com.
Does this form automatically submit to a database?
Not by itself. This is a true single HTML file, so there is no backend attached. The form currently generates a formatted message and can open the user’s email app. For full automatic storage, you would connect Formspree, Netlify Forms, or another backend later.
Can I still use this right away?
Yes. Payment is already integrated, and the intake flow is usable immediately. The only thing that is not fully automated is saving form submissions to a server.
What Stripe redirect URL should I set?
Set the success URL to https://readytohireu.netlify.app/?payment=success and the cancel URL to https://readytohireu.netlify.app/?payment=cancel.
